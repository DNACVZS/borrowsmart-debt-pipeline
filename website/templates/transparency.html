Transparency

Sources, formulas, responsible-use notes, and prototype limitations.

Data sources

  • FSA: ./data/raw/portfolio-by-location.xls
  • ACS: ./data/raw/ACSDT5Y2024.B19013-Data.csv
  • CFPB: ./data/raw/map_data_STU.csv

Product Context

  • Core output: state-level debt burden indicator (DTI proxy), risk label, and origination trend signal.
  • Who uses this: nonprofit financial advisors and program managers.
  • Decision supported: where outreach and counseling resources may need to be concentrated.
  • Geography: 50 U.S. states plus Washington, DC. Territories are excluded from this prototype.
  • Data level: state-level aggregates only, no individual records.
  • Platform scope: prototype site with local file processing, no external APIs, and no authentication.

How to Use This Tool

  • Use the dashboard to compare states, spot higher-burden regions, and prioritize advisor follow-up.
  • Treat risk labels as planning signals for outreach, not as borrower-level judgments.
  • Use this tool alongside local program knowledge, staffing constraints, and borrower context.

Metric definitions

  • average_debt_per_borrower = total_outstanding_balance / borrower_count
  • median_household_income = ACS B19013_001E
  • debt_to_income_ratio = average_debt_per_borrower / median_household_income
  • origination_yoy_change = CFPB value

Risk label thresholds

  • Low: debt_to_income_ratio < 0.45
  • Medium: 0.45 ≤ debt_to_income_ratio < 0.60
  • High: debt_to_income_ratio ≥ 0.60

Limitations

  • Aggregated state-level values only; no individual borrower-level context.
  • State averages may hide within-state differences.
  • This prototype supports program decisions and is not individualized financial advice.
